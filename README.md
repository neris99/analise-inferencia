Análise exploratória e inferencial utilizando Python, com foco na relação entre
sintomas depressivos (PHQ-9), atividade física e qualidade da dieta.

Projeto desenvolvido no Google Colab como atividade prática de Ciência de Dados.

## Dados
- DEMO_PHQ.csv: dados sociodemográficos e questionário PHQ-9
- PAG_HEI.csv: dados de atividade física (PAG) e qualidade da dieta (HEI)

## Análises realizadas
- Tratamento e limpeza dos dados
- Criação do score PHQ-9
- EDA univariada e bivariada
- Testes estatísticos:
  - Correlação de Pearson
  - Teste t independente
  - ANOVA
  - Qui-quadrado

## Ferramentas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Google Colab
